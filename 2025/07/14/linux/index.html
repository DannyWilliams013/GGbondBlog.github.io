一、磁盘分区

1.MSDOS（MBR） 与 GPT 磁盘分区表（partition table）

MBR磁盘分区表与格式：第一个扇区 （512Bytes）主要存储两个数据，分别是：

  • 开机记录区（Master Boot Record）MBR,有 446 Bytes
  • 分区表（partition table）：，有64 Bytes

MBR 有主要分区、延伸分区与逻辑分区的特性

2.根目录数结构

“挂载”就是利用一个目录当成进入点，将磁盘分区的数据放置在该目录下； 也就是
说，进入该目录就可以读取该分区的意思。
